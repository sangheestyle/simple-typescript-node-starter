# Simple TypeScript Node Starter

This base code is a minimized version of [TypeScript Node Starter](https://github.com/Microsoft/TypeScript-Node-Starter).

## Quick start

To setup

```
npm i
```

To run test

```
npm test
# or
npm t
```

To run server

```
npm start
```

Then

```
curl -X GET localhost:3000/api
```
